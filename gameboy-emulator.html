게임보이 에뮬레이터
.gb 또는 .gbc 롬 파일을 첨부하면
바로 이 자리에서 게임을 플레이할 수 있어요.
지원 형식: .gb / .gbc
파일은 이 기기(브라우저) 안에서만 실행되며 서버로 전송되지 않아요.
📚 내 게임 보관함
Nintendo GameBoy
BATTERY DOT MATRIX
게임보이 온라인
SELECT
START
현재 실행 중: -
💡 다른 기기에서 이어하려면: 이 기기에서 저장 내보내기로 파일을 받은 뒤, 다른 기기의 이 페이지에서 같은 롬을 먼저 불러오고 저장 가져오기로 그 파일을 선택하세요.
PC 키보드: 방향키 = 이동 · X/J = A · Z/Q = B · Shift = Select · Enter = Start
